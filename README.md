# pterminal

## Suckless Wayland Terminal Emulator
![screenshot](screenshot.png)

--------------------
Simple terminal emulator for Wayland which sucks less.  
Based on suckless st


Requirements
------------
In order to build st you need the Xlib header files.


Build
------------

    make
